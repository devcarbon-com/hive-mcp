org-clj Library

Native Clojure Parser

Why Native?

  • No elisp dependency needed
  • Immutable data structures
  • Schema validation with Malli
  • Easy to test (46+ assertions)

Usage

Parse ‚Üí Query ‚Üí Transform

(require '[hive-mcp.org-clj.parser :as p])
(require '[hive-mcp.org-clj.query :as q])

(def doc (p/parse-document (slurp "tasks.org")))

;; Find TODOs
(q/find-by-status doc "TODO")

;; Statistics
(q/task-stats doc)
;; => {:total 10 :todo 3 :done 7}

The Interaction Flow

  1. User Request ‚Üí Natural language prompt
  2. Claude Processing ‚Üí Selects appropriate MCP tools
  3. Tool Execution ‚Üí read_file, file_edit, eval_elisp
  4. Emacs Actions ‚Üí Buffer ops, export, browse-url
  5. Result ‚Üí Browser shows updated presentation
